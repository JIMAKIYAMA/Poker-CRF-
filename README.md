# Poker-CRF-
Implementação em Python puro do algoritmo CFR (Counterfactual Regret Minimization) aplicado ao Kuhn Poker. Resolve o Equilíbrio de Nash do zero.

O que é Kuhn Poker?
O Kuhn Poker é um jogo de soma zero extremamente simples: Possui 3 cartas (Rei, Dama, Valete), com 2 jogadores , sendo que cada um receba uma carta e o jogador pode escolher se passa ou aposta.
O código em python consegue calcular a porcentagem da aposta/passa, usando a teoria do jogos(O Equilíbrio de Nash), para isso, uso o algoritmo CFR (Counterfactual Regret Minimization).
